Key Ideas

  • Using a radar to detect displacement.
  • Bulk sampling of analog data.
  • Buffering to DataStore and subsequent streaming to SD card.
  • Wireless monitoring on another mote.

Description

This app note uses the BumbleBee micro-power radar to detect displacement of a person, animal, vehicle or other moving object while filtering out oscillating motion such as a bush in the wind. This can be used in a larger application such as home security or perimeter monitoring.

The app note includes options to log the sampling data and analysis results to an SD card. Using a second mote, the real-time status of the displacement detection can be monitored remotely at a distance.

Complexity Level

Moderately complex. It covers how the radar detects displacement, how to use DataStore as a buffer for subsequent streaming, and how to use the radio to monitor detection status.
